
To run the examples (Unix/Linux)
- update the variables in Makefile.config
- run 'make' to build the examples
- run 'cd <example_dir>; ./test' to run an example

To run the examples (Visual C++ 2005 IDE)
- Edit the paths in build-msvc\systemc.vsprops
- Load the project build-msvc\unit_test.sln

To run the examples (Visual C++ 2005/Visual C++ 2003 command line)
- Read the instructions in build-msvc\READMEcommandlinemsvc.txt
